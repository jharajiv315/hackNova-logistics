# hackNova-logistics

## Overview  
This repository holds the source code for *hackNova Logistics* — a demo logistic/tracking web application built with HTML, CSS, and JavaScript.

## Features  
- Responsive front-end showcasing pages like Home, Features, Tracking, Dashboard  
- Asset & script organisation under `assets/`, `scripts/`, `styles/` folders  
- Clean structure to support further expansion into a full logistics platform
